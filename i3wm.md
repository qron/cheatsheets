Mod key
-------

	Cmd/Win or Alt

Basis
-----

| Function           | Shortcut          |
| ------------------ | ----------------- |
| Restart i3wm       | `MOD + Shift + r` |
| Reload i3wm config | `MOD + Shift + c` |
| Open terminal      | `MOD + Enter`     |
| Open dmenu         | `MOD + d`         |
| Quit curent window | `MOD + Shift + q` |

Workspaces
----------

### Move current window to workspace

	MOD + Shift + WORKSPACE_SHORTCUT
	
| Target | Shortcut            |
| ------ | ------------------- |
| 1      | `MOD + Shift + 1`   |
| N      | `MOD + Shift + N`   |
| 10     | `MOD + Shift + 0`   |

### Go to workspace

	MOD + WORKSPACE_SHORTCUT
	
| Target | Shortcut    |
| ------ | ----------- |
| 1      | `MOD + 1`   |
| N      | `MOD + N`   |
| 10     | `MOD + 0`   |

### Mode

| Mode            | Shortcut              | Toogle |
| --------------- | --------------------- | ------ |
| Fullscreen      | `MOD + f`             | Yes    |
| Float           | `MOD + Shift + Space` | Yes    |
| Stack           | `MOD + s`             | No     |
| Tab             | `MOD + w`             | No     |
| Split           | `MOD + e`             | No     |
| Vertical Tile   | `MOD + v`             | No     |
| Horizontal Tile | `MOD + h`             | No     |
| Resize          | `MOD + r`             | No     |

#### Resizing

| Function         | Shortcut    |
| ---------------- | ----------- |
| Exit resize mode | `Escape`    |
| Increase height  | `↓ or k`    |
| Decrease height  | `↑ or l`    |
| Increase width   | `→ or ;`    |
| Decrease width   | `← or j`    |

### Navigate through windows/tabs/stacks

| Direction | Shortcut           |
| --------- | ------------------ |
| Top       | `MOD + [ ↑ or l ]` |
| Bottom    | `MOD + [ ↓ or k ]` |
| Left      | `MOD + [ ← or j ]` |
| Right     | `MOD + [ → or ; ]` |

### Move current windows

| Direction | Shortcut                   |
| --------- | -------------------------- |
| To top    | `MOD + Shift + [ ↑ or l ]` |
| To bottom | `MOD + Shift + [ ↓ or k ]` |
| To left   | `MOD + Shift + [ ← or j ]` |
| To right  | `MOD + Shift + [ → or ; ]` |
